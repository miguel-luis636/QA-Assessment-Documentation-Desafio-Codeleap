# 📱 Template App - QA Assessment

## 📌 Sobre o Projeto

Este repositório documenta uma avaliação prática de QA , com o objetivo de demonstrar habilidades em testes de software em um cenário realista.

A aplicação utilizada é uma versão de testes do app **Template**, desenvolvido internamente pela CodeLeap. Trata-se de um ambiente não produtivo, criado exclusivamente para fins de avaliação técnica.

O foco principal deste projeto é evidenciar:

* Capacidade de identificar e reportar bugs
* Clareza e organização na documentação
* Abordagem de testes e priorização
* Atenção à experiência do usuário (UX)

---

## 🧠 Objetivo da Avaliação

Realizar testes exploratórios na aplicação, identificando:

* Bugs funcionais
* Problemas de interface (UI)
* Falhas de usabilidade (UX)
* Comportamentos inconsistentes

Todos os achados devem ser documentados de forma clara e objetiva.

---

## 📱 Visão Geral da Aplicação

O **Template App** é uma aplicação base utilizada como fundação para projetos de clientes, contendo componentes reutilizáveis, telas comuns e utilidades diversas.

Para esta avaliação, o foco está no comportamento da aplicação sob a perspectiva do usuário final, sem necessidade de conhecimento da arquitetura interna.

---

## 🚀 Funcionalidades Principais

### 📰 Social Feed

* Criação e publicação de posts
* Curtir e favoritar conteúdos
* Interação com posts
* Navegação e rolagem do feed

### 💬 Chat

* Envio e recebimento de mensagens
* Criação e navegação entre conversas

### 👤 Perfil

* Visualização de informações do usuário
* Edição de dados (nome, foto, etc.)

---

## 🧪 Estratégia de Testes

Os testes foram conduzidos com foco em:

* Testes exploratórios
* Validação de fluxos principais
* Testes de usabilidade
* Testes de inputs (válidos e inválidos)
* Análise de comportamento da interface

---

## 🎯 Áreas Prioritárias

Durante a execução dos testes, foi dada prioridade para:

1. **Social Feed** — criação e interação com posts
2. **Chat** — envio de mensagens e navegação
3. **Perfil** — edição e persistência de dados
